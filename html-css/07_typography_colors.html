@font-face

Används av Google fonts

Vi kan även skapa egna @font-face

@font-face {
  font-family: 'Bungee';
  font-style: normal;
  font-weight: 400;
  /* If installed, used installed, else load from server */
  src: local('Bungee'), local('Bungee-regular'), url(Bungee.woff2) format('woff2');
}

Problem med webfonts

FOUT

Flash of Invisible Text: Default browser behavior to render text invisible while the web font is loading. In modern browsers, FOIT lasts a maximum of three seconds

FOIT - Flash of invisible Text

Flash of Unstyled Text: Default behavior to render text with the fallback system font while the web font has loaded

Zach Leatherman - FOIT vs. FOUT Demo

A Comprehensive Guide To Font Loading Strategies

Bra lösningar med sämre stöd

font-display: swap;

Sätt i @font-face för att automatisk byta till din häftiga webfont när den är tillgänglig

Gradients

Linear gradient

Kräver minst två color stops

background: linear-gradient(lightsalmon, teal);

Default är top -> bottom

Kan också specificera riktning

background: linear-gradient(to right, lightsalmon, teal);

Kan också specificera riktning

background: linear-gradient(to left, lightsalmon, teal);

Riktning med grader ger mer möjligheter

background: linear-gradient(45deg, lightsalmon, teal);

Sharp color stops

background: 
linear-gradient(lightsalmon 40%, teal 40%, teal 60%, lightsalmon);

Radial gradient
