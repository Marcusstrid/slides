Keyframe Animation

Vi kan definiera egna animationer förutom de inbyggda

@keyframes

@keyframes

@keyframes myAnimation { 
  from{
    opacity: 0;
  }
  to{
    opacity: 1;
  }
}

I princip en transition, men vi kan t.ex. loopa övergången

@keyframes myAnimation { 
  from{
    opacity: 0;
    background: teal;
  }
  to{
    opacity: 1;
    background: slategray;
  }
}

Vi får animera flera css-egenskaper

@keyframes myAnimation { 
  0%{
    opacity: 0;
  }
  100%{
    opacity: 1;
  }
}

Vanligast är att ange steg i %

@keyframes myAnimation { 
  0%{
    opacity: 0;
  }
  10% {
    opacity: 0.2;
  }
  100%{
    opacity: 1;
  }
}

2.3.1 Three Flashes or Below Threshold

Web pages do not contain anything that flashes more than three times in any one second period, or the flash is below the general flash and red flash thresholds. (Level A)

.box {
  animation-name: myAnimation; /* previous animation */
  animation-duration: 2s; /* play for 2s */
  animation-delay: 1s;    /* start after 1s */
  animation-iteration-count: infinite; /* go on forever */
  animation-direction: normal;  /* do not reverse */
  animation-timing-function: ease; /* ease the end and start */
  animation-fill-mode: none; /* do not apply styles */
  animation-play-state: running; /* start it */
}
