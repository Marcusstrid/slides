CSS-ramverk

Vad är ett ramverk?

A framework is a skeleton where the application defines the "meat" of the operation by filling out the skeleton. The skeleton still has code to link up the parts but the most important work is done by the application.
https://stackoverflow.com/a/148759/5836872

Största anledningen till att jag ofta drar in ett ramverk

FORMS / INPUTS

I nästan allt finns något sorts inputfält som är jävla jobbigt att styla

Forms

Label + input grupperas med en div med klassen form-group

Själva inputfältet stylas med klassen form-control

Knappar stylas med btn

https://getbootstrap.com/docs/4.0/components/forms/ 
<form>
  <div class="form-group">
    <label for="input">Input</label>
    <input 
      type="input"
      class="form-control" 
      id="input"
      placeholder="Enter input">
  </div>
  <button type="submit" class="btn btn-primary">Submit</button>
</form>
